Lab 2 — Recursive Factorial
  Code: defines factorial(n) with a base case (n==0 or n==1 → 1) and the

  learners see multiple results at once.

Lab 3 — Default Arguments
  Code: defines power(base, exponent=2). Three calls demonstrate:
    1. Calling with one argument → default exponent used (3² = 9)
    2. Overriding the default    (3³ = 27)
    3. A larger exponent         (2¹⁰ = 1024)
  Inline comments on each call explain what is happening.
